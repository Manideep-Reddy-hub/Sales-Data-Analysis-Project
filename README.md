# 🛒 Superstore Sales Data Analysis

## 📌 Project Overview

This project analyzes a retail superstore dataset to uncover **sales performance, profitability, and operational efficiency insights**.

The focus is not just on reporting data, but on identifying **business problems and recommending actionable solutions**.

---

## 🎯 Objectives

* Identify **loss-making products**
* Analyze **regional performance and efficiency**
* Evaluate **profit margins**
* Understand **sales trends over time**
* Assess **shipping performance**

---

## 🛠️ Tools Used

* Python
* Pandas
* Matplotlib

---

## ▶️ How to Run

```bash
python superstore.py superstore.csv
```

---

## 📊 Key Analysis

### 1. Data Preparation

* Handled missing values
* Converted date columns
* Created new features:

  * Month
  * Year
  * Delivery Days

---

### 2. Exploratory Analysis

* Sales vs Profit by Sub-Category
* Region-wise Sales and Profit
* Monthly Sales Trends
* Shipping Efficiency

---

### 3. Advanced Analysis

* **Profit Margin by Region**

  * Measures how efficiently regions convert sales into profit
* Identification of **loss-making products**
* Detection of **high-sales but low-profit categories**

---

## 📈 Visualizations

* Sales vs Profit by Sub-Category (loss highlighted in red)
* Region-wise performance comparison
* Shipping mode efficiency

---

## 🔍 Key Insights

* Some sub-categories generate **negative profit despite high sales**, indicating pricing or discount issues
* The **top-performing region generates the highest profit**, showing strong market demand
* The **highest profit margin region is the most efficient**, not necessarily the highest sales region
* Sales show **seasonal trends**, useful for planning inventory and campaigns
* Shipping performance varies, affecting customer experience

---

## 💡 Business Recommendations

* Adjust pricing or discounts for **loss-making products**
* Invest more in **high-performing and high-margin regions**
* Improve **low-margin regions** by reducing costs or optimizing pricing
* Optimize **slow shipping methods**
* Use seasonal trends for **demand forecasting**
